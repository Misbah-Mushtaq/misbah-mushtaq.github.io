Release Notes
v1.1.5 12 Jul 2026

Custom Tasbeeh Targets & Prayer Time Format

New Features

  • My Tasbeeh — create personal recitation goals with a name, optional Arabic text, and target count (e.g. Al-Noor × 25,000). Progress tracked independently and persisted across sessions.
  • Counting view — dedicated ring display with Completed / Remaining / Progress stat cards and green completion state.
  • Tasbih is now the default tab — Tasbih opens first; Sessions is the second tab.

Bug Fixes

  • Prayer times follow device 12/24h setting — a native Android module now reads DateFormat.is24HourFormat() instead of locale defaults. The change propagates to prayer cards, next prayer label, optional prayers, and the home screen widget.
  • Widget highlights current prayer — previously highlighted the next prayer.
  • Swipe to delete notifications — swipe left on any notification row to reveal a delete action.
  • Ghost "Noor" empty notifications no longer appear in inbox.
  • Fixed duplicate key warning in the notification list.
  • Fixed left-side icon gap in notifications screen.
  • Fixed @expo/ui and expo-widgets version ranges causing CI failures.

v1.1.4 Jun 2026

iOS Widgets & In-App Notification Inbox

New Features

  • iOS widgets — Prayer Times, Islamic Date, Daily Verse, and Dhikr Counter widgets for the iPhone home and lock screen via WidgetKit.
  • Notification inbox — bell icon opens a history of all received notifications, grouped by Today / Earlier with filter chips by type.
  • Deep-link from notifications — tapping a notification navigates directly to the relevant section of the app.
  • Background notification sync — missed notifications captured via background fetch task and notification ledger.

Bug Fixes

  • Fixed iOS build number display in Settings.
  • Resolved Reanimated strict-mode warnings in SkeletonLoader, SurahBanner, and Qibla screen.
  • Removed expo-widgets from Android plugin list — iOS-only package was breaking Android builds.
  • Throttled notification reschedule on app open to protect imminent alarms.

v1.1.0 May 2026

Zakat Calculator & Firebase Analytics

New Features

  • Zakat Calculator — full nisab calculation with live gold/silver prices, asset entry, and disbursement tracking.
  • Edit & recalculate — existing Zakat records can be reopened and updated.
  • Firebase Analytics — anonymous screen and event tracking.

Bug Fixes

  • Fixed race condition when switching metal type during live price fetch.
  • Fixed keyboard covering disbursement inputs on smaller screens.
  • Fixed Live Price button overflow on narrow devices.
